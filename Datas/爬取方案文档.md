# 电力相关标准数据爬取方案文档

## 一、项目概述

爬取电力相关的国家标准和行业标准数据，包括基础信息和详情信息，并导出为Excel文件。

**数据来源分类：**
- 国家标准接口1：std.samr.gov.cn（GB搜索API）
- 国家标准接口2：openstd.samr.gov.cn（国家标准全文公开系统）
- 行业标准接口：hbba.sacinfo.org.cn（行业标准信息服务平台）

---

## 二、数据来源详情

### 数据来源1：国家标准接口1（std.samr.gov.cn）

#### 1.1 列表数据接口

**接口地址：**
```
https://std.samr.gov.cn/gb/search/gbQueryPage
```

**请求方式：** GET

**请求参数：**

| 参数名 | 说明 | 示例值 |
|--------|------|--------|
| searchText | 搜索关键词 | 电力 |
| ics | 国际标准分类号（可选） | 空 |
| state | 标准状态（可选） | 空 |
| ISSUE_DATE | 发布日期范围（可选） | -36（近36个月） |
| sortOrder | 排序方式 | asc |
| pageSize | 每页记录数 | 50 |
| pageNumber | 页码 | 1 |
| _ | 时间戳防缓存 | 1774853445462 |

**请求示例：**
```
https://std.samr.gov.cn/gb/search/gbQueryPage?searchText=电力&ics=&state=&ISSUE_DATE=-36&sortOrder=asc&pageSize=50&pageNumber=1&_=1774853445462
```

**响应格式：** JSON

**响应结构：**
```json
{
  "total": 95,
  "pageNumber": 1,
  "rows": [...]
}
```

**rows 单条记录字段：**

| 字段名 | 说明 | 示例值 |
|--------|------|--------|
| id | 记录唯一标识 | 28045D84D9829055E06397BE0A0A7C5C |
| C_STD_CODE | 标准号 | GB/T 1094.7-2024 |
| C_C_NAME | 标准中文名称 | 电力变压器 第7部分：油浸式电力变压器负载导则 |
| ISSUE_DATE | 发布日期 | 2024-11-28 |
| ACT_DATE | 实施日期 | 2025-06-01 |
| STATE | 标准状态 | 现行 |
| STD_NATURE | 标准性质 | 推荐性 |

#### 1.2 详情页数据接口

**详情页地址格式：**
```
https://std.samr.gov.cn/gb/search/gbDetailed?id={id}
```

**详情页提取信息：**

| 信息类型 | 说明 | 示例 |

| 归口单位 | 技术归口单位 | 全国变压器标准化技术委员会 |
| 主管部门 | 主管部门名称 | 中国电器工业协会 |
| 起草单位 | 起草单位列表 | 沈阳变压器研究院有限公司 |

**请求示例：**

https://openstd.samr.gov.cn/bzgk/gb/std_list?r=0.783&page=1&pageSize=50&p.p1=0&p.p2=电力&p.p7=3&p.p90=circulation_date&p.p91=desc

## 四、爬取流程

### 国家标准接口1流程
```
1. GET请求gbQueryPage接口 → 解析JSON获取total计算总页数
2. 循环获取所有页面数据 → 合并rows列表
3. 对每条记录：
   - 提取id拼接详情链接
   - 提取基础字段（标准号、名称、日期、状态、性质）
   - GET请求详情页提取摘要信息
4. 整合数据导出Excel文件
```

### 国家标准接口2流程
```
1. GET请求std_list接口 → 解析HTML表格获取数据
2. 从onclick属性提取hcno值
3. 循环分页获取所有数据
4. 对每条记录：
   - 提取基础字段（序号、标准号、名称、类别、状态、日期）
   - GET请求详情页newGbInfo提取详细信息
5. 整合数据导出Excel文件
```

### 行业标准流程
```
1. POST请求stdQueryList接口 → 解析JSON获取total计算总页数
2. 循环POST请求所有页面数据 → 合并records列表
3. 时间戳转换（issueDate/actDate为毫秒时间戳）
4. 对每条记录：
   - 提取pk拼接详情链接
   - 提取基础字段（标准号、名称、日期、状态、行业）
   - GET请求详情页stdDetail提取详细信息
5. 整合数据导出Excel文件
```

## 七、注意事项

1. **请求频率控制**：每次请求间隔0.3-0.5秒，避免服务器压力
2. **HTML标签清理**：部分字段可能包含HTML标签，需清理
3. **时间戳转换**：行业标准接口的日期为毫秒时间戳，需转换为日期格式
4. **版权限制**：部分采标国家标准因版权问题无法下载全文
5. **空值处理**：部分新发布标准实施日期可能尚未确定
