### v1.2 - 2026-03-31 (第3轮反馈)

#### 反馈内容

4. **数据看板优化**

5. **页面设计统一**
   - 减少圆角，统一为 2px

#### 爬虫脚本说明

**国标脚本** (合并接口1+2):
- 接口1: `std.samr.gov.cn/gb/search/gbQueryPage` - JSON API
- 接口2: `openstd.samr.gov.cn/bzgk/gb/std_list` - HTML解析
- 合并逻辑: 以接口2字段为主，接口1补充

**行标脚本**:
- 接口: `hbba.sacinfo.org.cn/stdQueryList` - POST JSON
